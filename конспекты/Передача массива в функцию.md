# Передача массива в функцию (C++)

## Основная идея

В C++ при передаче массива в функцию он **не копируется**. Передаётся указатель на первый элемент.

```cpp
void func(int arr[]) 
{
    // arr — это указатель int*
}
```

Эквивалентные формы:

```cpp
void func(int arr[]);
void func(int* arr);
```

## Передача массива

```cpp
void print(int arr[], int size) 
{
    for (int i = 0; i < size; i++) 
    {
        std::cout << arr[i] << " ";
    }
}

int main() 
{
    int a[3] = {1, 2, 3};
    print(a, 3);
}
```

## Почему нужен размер массива

После передачи массива в функцию теряется информация о его длине.

```cpp
void func(int arr[]) 
{
    // нельзя узнать размер через sizeof(arr)
}
```

## Передача через указатель

То же самое, что и массив:

```cpp
void sum(int* arr, int size) 
{
    int s = 0;
    for (int i = 0; i < size; i++) 
    {
        s += arr[i];
    }
}
```

## Изменение массива в функции

Функция работает с оригинальным массивом

```cpp
void change(int arr[], int size) 
{
    arr[0] = 100;
}

int main() 
{
    int a[3] = {1, 2, 3};
    change(a, 3);
    // a[0] == 100
}
```

## Передача многомерных массивов

Размер второй размерности обязателен

```cpp
void print(int arr[][3], int rows) 
{
    for (int i = 0; i < rows; i++) 
    {
        for (int j = 0; j < 3; j++) 
        {
            std::cout << arr[i][j] << " ";
        }
    }
}
```

## Альтернатива: std::array

```cpp
#include <array>

void func(std::array<int, 3> arr) 
{
    // копия массива
}
```

## Альтернатива: std::vector (рекомендуется)

```cpp
#include <vector>

void func(std::vector<int>& arr) 
{
    arr[0] = 10;
}
```

## Основные особенности

* Массив передаётся как указатель
* Размер массива не передаётся автоматически
* Изменения внутри функции влияют на оригинал

## Частые ошибки

### 1. Попытка узнать размер через sizeof

```cpp
void func(int arr[]) 
{
    int size = sizeof(arr); // ошибка: это размер указателя
}
```

### 2. Выход за границы массива

```cpp
arr[1000] = 5; // может привести к ошибке
```

## Итог

* Массив = указатель при передаче в функцию
* Всегда передавай размер отдельно
* Лучше использовать std::vector для безопасности
