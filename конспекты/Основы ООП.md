# Основы ООП

## Что такое ООП

ООП (объектно-ориентированное программирование) — подход, где программа строится из **объектов**, объединяющих данные и методы.

---

## Основные принципы ООП

### 1. Инкапсуляция

Сокрытие данных и доступ к ним только через методы.

```cpp
class Person 
{
private:
    int age;

public:
    void setAge(int a) 
    {
        age = a;
    }

    int getAge() 
    {
        return age;
    }
};
```

---

### 2. Наследование

Создание нового класса на основе существующего.

```cpp
class Animal 
{
public:
    void eat() 
    {
        std::cout << "Eating";
    }
};

class Dog : public Animal 
{
public:
    void bark() 
    {
        std::cout << "Bark";
    }
};
```

---

### 3. Полиморфизм

Один интерфейс — разные реализации.

#### Через виртуальные функции:

```cpp
class Animal 
{
public:
    virtual void sound() 
    {
        std::cout << "Animal sound";
    }
};

class Cat : public Animal 
{
public:
    void sound() override 
    {
        std::cout << "Meow";
    }
};
```

```cpp
Animal* a = new Cat();
a->sound(); // Meow
```

---

### 4. Абстракция

Скрытие деталей реализации, оставляя только важное.

```cpp
class Shape 
{
    public:
        virtual void draw() = 0; // чисто виртуальная функция
};

class Circle : public Shape 
{
public:
    void draw() override 
    {
        std::cout << "Circle";
    }
};
```

## Классы и объекты

### Класс

Шаблон для создания объектов.

### Объект

Конкретный экземпляр класса.

```cpp
class Car 
{
public:
    int speed;
};

Car c;
c.speed = 100;
```

---

## Конструкторы

Используются для инициализации объектов.

```cpp
class Car 
{
public:
    int speed;

    Car(int s) 
    {
        speed = s;
    }
};

Car c(100);
```

---

## Деструктор

Вызывается при удалении объекта.

```cpp
class Car 
{
public:
    ~Car() 
    {
        std::cout << "Destroyed";
    }
};
```

---

## Модификаторы доступа

* `public` — доступен всем
* `private` — доступ только внутри класса
* `protected` — доступ в классе и наследниках

---

## this указатель

Указывает на текущий объект.

```cpp
class A 
{
public:
    int x;

    void setX(int x) 
    {
        this->x = x;
    }
};
```

---

## Итог

* ООП = объекты + классы
* 4 принципа: инкапсуляция, наследование, полиморфизм, абстракция
* Используются классы, методы, конструкторы
