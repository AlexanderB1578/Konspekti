# Понятие классов

## Что такое класс

Класс — это пользовательский тип данных, который объединяет:

* данные (поля)
* функции (методы)

Он описывает структуру объектов.

---

## Объект класса

Объект — это экземпляр класса.

```cpp
class Car 
{
    public:
        int speed;
};

int main() 
{
    Car c1;
    c1.speed = 100;
}
```

---

## Объявление класса

```cpp
class Person 
{
    public:
        int age;
        std::string name;
};
```

---

## Поля класса

Переменные внутри класса.

```cpp
class A 
{
    public:
        int x;
        double y;
};
```

---

## Методы класса

Функции внутри класса.

```cpp
class Person 
{
    public:
        void sayHello() 
    {
        std::cout << "Hello";
    }
}
```

---

## Доступ к членам класса

```cpp
Person p;
p.age = 20;
p.sayHello();
```

---

## Модификаторы доступа

### public

Доступно извне класса

### private

Доступно только внутри класса

### protected

Доступно в классе и наследниках

```cpp
class A 
{
    private:
        int secret;

    public:
        int open;
}
```

---

## Инкапсуляция

Скрытие данных и доступ через методы.

```cpp
class Person 
{
    private:
        int age;

    public:
        void setAge(int a) 
        {
            age = a;
        }

        int getAge() 
        {
            return age;
        }
};
```

---

## Разделение на .h и .cpp (идея)

### Header (.h)

```cpp
class Car 
{
    public:
        void drive();
};
```

### Implementation (.cpp)

```cpp
#include "Car.h"

void Car::drive() 
{
    std::cout << "Driving";
}
```

---

## this указатель

Указывает на текущий объект

```cpp
class A 
{
    public:
        int x;

        void setX(int x) 
    {
        this->x = x;
    }
};
```

---

## Размер объекта

Объект занимает память под все поля класса.

```cpp
class A 
{
    int x;
    int y;
};
```

---

## Итог

* Класс = шаблон объекта
* Объект = экземпляр класса
* Есть поля и методы
* Доступ регулируется public/private/protected
* Основа ООП в C++
