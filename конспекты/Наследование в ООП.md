# Наследование в ООП (C++)

## Что такое наследование

Наследование — это механизм, при котором новый класс (потомок) получает свойства и методы существующего класса (родителя).

---

## Базовый синтаксис

```cpp
class Animal 
{
    public:
        void eat() 
        {
            std::cout << "Eating";
        }
}

class Dog : public Animal 
{
    public:
        void bark() 
        {
            std::cout << "Bark";
        }
};
```

---

## Типы наследования

```cpp
class A : public B {};      // public наследование
class A : protected B {};   // protected наследование
class A : private B {};     // private наследование
```

### Кратко:

* public — сохраняет доступы
* protected — public становится protected
* private — всё становится private

---

## Доступ к членам базового класса

| В базовом | public наследование | protected наследование | private наследование |
| --------- | ------------------- | ---------------------- | -------------------- |
| public    | public              | protected              | private              |
| protected | protected           | protected              | private              |
| private   | недоступен          | недоступен             | недоступен           |

---

## Переопределение методов (override)

```cpp
class Animal 
{
    public:
        virtual void sound() 
    {
        std::cout << "Animal sound";
    }
};

class Cat : public Animal 
{
    public:
        void sound() override 
        {
            std::cout << "Meow";
        }
};
```

---

## Виртуальные функции

Позволяют реализовать полиморфизм.

```cpp
Animal* a = new Cat();
a->sound(); // Meow
```

## Конструкторы при наследовании

Сначала вызывается конструктор базового класса, затем потомка.

```cpp
class A 
{
    public:
        A() 
        {
            std::cout << "A\n";
        }
};

class B : public A 
{
    public:
        B() 
        {
            std::cout << "B\n";
        }
};
```

---

## Вызов конструктора базового класса

```cpp
class A 
{
    public:
        A(int x) {}
};

class B : public A 
{
    public:
        B() : A(10) {}
};
```

---

## protected

Доступен в базовом классе и наследниках

```cpp
class A 
{
    protected:
        int x;
};

class B : public A 
{
    public:
        void setX(int v) 
        {
            x = v;
        }
};
```

---

## Множественное наследование

```cpp
class A 
{
    public:
        void a() {}
};

class B 
{
    public:
        void b() {}
};

class C : public A, public B {};
```

---

## Проблема ромба

Возникает при множественном наследовании.
Решается виртуальным наследованием.

```cpp
class A {};
class B : virtual public A {};
class C : virtual public A {};
class D : public B, public C {};
```

---

## Итог

* Наследование = расширение класса
* Есть базовый и производный класс
* Управляется доступом (public/protected/private)
* Важно для повторного использования кода
* Основа полиморфизма
