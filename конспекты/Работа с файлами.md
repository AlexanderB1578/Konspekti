# Работа с файлами в C++

## Основные заголовки

Для работы с файлами используется библиотека:

```cpp
#include <fstream>
```

В ней есть три основных класса:

* `ifstream` — чтение из файла
* `ofstream` — запись в файл
* `fstream` — чтение и запись

---

## Запись в файл

```cpp
#include <fstream>

int main() 
{
    std::ofstream file("data.txt");

    if (file.is_open()) 
    {
        file << "Hello, file!" << std::endl;
        file << 123 << std::endl;
    }

    file.close();
}
```

---

## Чтение из файла

```cpp
#include <fstream>
#include <iostream>
#include <string>

int main() 
{
    std::ifstream file("data.txt");
    std::string line;

    if (file.is_open()) 
    {
        while (std::getline(file, line)) 
        {
            std::cout << line << std::endl;
        }
    }

    file.close();
}
```

---

## Открытие файла

```cpp
file.open("data.txt");
```

Проверка:

```cpp
if (!file) 
{
    std::cout << "Ошибка открытия файла";
}
```

---

## Режимы открытия файла
 Режимы:           
 * `ios::in` - чтение             
 * `ios::out` - запись             
 * `ios::app` - добавление в конец 
 * `ios::trunc` - очистка файла      
 * `ios::binary` - бинарный режим     

### Пример:

```cpp
std::ofstream file("data.txt", std::ios::app);
```

---

## Чтение по словам

```cpp
std::string word;
while (file >> word) 
{
    std::cout << word << std::endl;
}
```

---

## Запись в бинарный файл

```cpp
struct Data 
{
    int x;
    float y;
};

Data d = {10, 3.14f};

std::ofstream file("data.bin", std::ios::binary);
file.write((char*)&d, sizeof(d));
file.close();
```

---

## Чтение бинарного файла

```cpp
Data d;

std::ifstream file("data.bin", std::ios::binary);
file.read((char*)&d, sizeof(d));
file.close();
```

## Частые ошибки

### 1. Не проверили открытие файла

```cpp
std::ifstream file("no.txt");
// файл может не открыться
```

### 2. Не закрыли файл

```cpp
// файл лучше закрывать или использовать RAII
```

### 3. Перезапись файла

```cpp
std::ofstream file("data.txt"); // стирает содержимое
```

---

## Итог

* `ifstream` — чтение
* `ofstream` — запись
* `fstream` — оба режима
* Всегда проверяй открытие файла
* Используй режимы (`app`, `binary`, `trunc`) по необходимости
