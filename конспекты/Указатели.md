# Указатели в C++

## Что такое указатель

Указатель — это переменная, которая хранит адрес другой переменной в памяти.

```cpp
int a = 10;
int* p = &a; // p хранит адрес a
```

## Основные операции

**1. Получение адреса** 

```cpp
int a = 5;
int* p = &a;
```

**2. Разыменование**

Позволяет получить значение по адресу.

```cpp
int a = 5;
int* p = &a;
int value = *p; // value = 5
```

## Изменение значения через указатель

```cpp
int a = 5;
int* p = &a;
*p = 20; // a теперь равно 20
```

## Нулевой указатель

Указатель, не указывающий ни на какой объект.

```cpp
int* p = nullptr;
```

## Указатели и массивы

Имя массива — это указатель на первый элемент.

```cpp
int arr[3] = {1, 2, 3};
int* p = arr;

std::cout << *p;     // 1
std::cout << *(p+1); // 2
```

## Арифметика указателей

Указатели можно увеличивать/уменьшать.

```cpp
int arr[3] = {1, 2, 3};
int* p = arr;

p++; // теперь указывает на arr[1]
```

## Указатели и функции

### Передача по указателю

```cpp
void increment(int* x) 
{
    (*x)++;
}

int main() 
{
    int a = 5;
    increment(&a);
    // a = 6
}
```

## Указатель на указатель

```cpp
int a = 10;
int* p = &a;
int** pp = &p;
```

## Динамическая память

### Выделение (new)

```cpp
int* p = new int(5);
```

### Освобождение (delete)

```cpp
delete p;
p = nullptr;
```

## Частые ошибки

* Разыменование nullptr

```cpp
int* p = nullptr;
//*p; // ошибка
```

* Утечки памяти

```cpp
int* p = new int(10);
// забыли delete
```

* Висячий указатель

```cpp
int* p = new int(5);
delete p;
// p теперь указывает на освобождённую память
```

## Итог

* Указатель хранит адрес
* `*` — доступ к значению
* `&` — получение адреса
* Важно контролировать память (new/delete)
